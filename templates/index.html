Welcome!

This user guide is designed to be a working desk reference for GOLDesp Users, and should supplement the formal classroom training received by new users. This version of the guide is based on the following version of GOLDesp:

  • Integrated Supply & Maintenance
  • Version 17.7

To use this guide, select the function that you need help with in the collapsable menu on the left hand side of the page. Many options have sub menus that will take you into more specific help topics.

Walktrough Guides

The link at the bottom of the menu will take you to the interactive 'Process Walkthroughs' tool, which will guide you through common user processes step-by-step (for example, opening a work order).

Depending on the options selected by your organization when purchasing GOLDesp, these may be either generic or specific to your business. See the notes at the start of the tool for more details.

Further Help

Further help can be found from the following sources:

  • GOLDesp Online Help
  • Unit Champion User
  • Your organization's GOLDesp Training Section
