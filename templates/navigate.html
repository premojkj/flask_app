Navigating GOLDesp

GOLDesp is designed to be easy to navigate by using a combination of menus, lookups, dropdowns, links and various other tools standard to Web-based design.

Please note: GOLDesp is a Commercial Off-The-Shelf (COTS) application, so the functionality and appearance of screens may be customized for your organization. Our Online Help includes all current GOLDesp functionality. Therefore, what you see on your screen may not reflect exactly what is described in the Online Help. In addition, of course, some elements of GOLDesp may not be visible if you do not have the appropriate level of security.

Menu Bar

Within GOLDesp, the Global Navigation line, known as the menu bar, serves as a main menu to navigate between functional areas.

The User may click on any item listed in the menu bar. When clicked, the navigation pane in GOLDesp displays the primary screen (also known as the User desktop ) for that functional area.

User Desktop

The User Desktop lets the User access functions for a selected Menu option. For example, the figure below illustrates some of the Menu selections that are likely to be available from the Enterprise menu.

The options displayed in the User Desktop depends on the User 's access to the applicable functions in the Main Menu. Access is controlled by the User Roles associated with the User ID of the logged-on User in GOLDesp.
To access a User Desktop function, click on the selection. GOLDesp displays the appropriate screen.

If you routinely use a particular functional area on logging on to GOLDesp, instead of displaying the “Home” pane by default, when you navigate to your preferred area you may click the “Make Default Desktop” link at the bottom of the navigation pane to make that area your personal default.

Home Desktop

The “Home” desktop is configurable by the GOLDesp Administrator. The right side provides several options that can be tailored for the User.
The GOLDesp Administrator should contact Tapestry Solutions Technical Support team for assistance.
The default Home Desktop provides the elements described below.

Search Online Help
Enter a word or words in the textbox, and click the arrow. The GOLDesp Help screen displays, and displays the Help topics containing those words.
Top Tips
The GOLDesp Administrator can configure the section with up to five entries, to display pop-up windows containing useful text, or links to external web pages.
For assistance, the Administrator should consult Tapestry Customer Support.

Saved Queries
The Saved Query functionality is available on a select group of Web 1.0 Inquiry screens throughout the application . This allows the User to enter values in one or more of the Criteria fields and save that data to a named query for subsequent recall. This section of the Home Desktop displays the queries that have been saved by the logged-on User.
