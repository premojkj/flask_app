Compass Calibration Inquiry

To carry out an inquiry:

  1. Navigate to Aircraft Data > Inquiries > Compass Calibration
  2. Enter the relevant search terms in the Compass Calibration Criteria screen.
  3. Select Search and your results will be displayed on the Compas Calibration List screen
  4. To open a record, click on View icon in the action column. The record will be open in Read Only Mode
  5. Users with the appropriate permisisons will have an Update button available in the left menu

Create Compass Calibration Record

To create a new compass calibration record:

  1. Navigate to Aircraft Data > Inquiries > Comapss Calibration
  2. Users with the appropriate permissions will have a Create New button available in the left menu. CLick on this button.
  3. The Compass Calibration Maintenance initial screen is presented. Complete the relevant fields and select Next Note: The following fields are mandatory:
    Serial
    - The serial number of the aircraft the record is being created for
    Part
    - The part number of the aircraft the record is being created for
    Work Order Number
    - The work order number that the swing is being recorded on
    Swing Commenced On
    - The date the swing was commenced
    Swing Completed On
    - The date the swing was completed
    Swing Location
    - The location of the swing. Must be a valid Site Code
    Swing Reason
    - Freetext field to enter reason for swing (i.e. annual check, post computer change etc)
    Compass Serial
    - The serial number of the compass being calibrated
  4. The Comapss Calibration Maintenance - Main Compass 1 screen is presented. Enter all details and readings for Main Compass 1
